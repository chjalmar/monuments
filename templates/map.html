API de Patrimonio de Santiago del Consejo Nacional de Monumentos de Chile

Propósito

Con la intención de hacer un aporte a la ciudad de Santiago y sus habitantes en la proximidad del Día del Patrimonio, he creado un pequeño API para acceder a la información del patrimonio de la ciudad disponible en www.monumentos.cl para la Región Metropolitana. ¡Espero este pequeño aporte nos ayude a disfrutar más de la ciudad!

Descripción y Herramientas

El API se encuentra publicado en la dirección:

https://intense-shelf-84410.herokuapp.com/

Y devuelve un objeto en formato JSON con todos los monumentos disponibles en la página para la Región Metropolitana. Para cada monumento el objeto contiene, según esté o no disponible la información en la Web del CNM:

  1. - ID
  2. - Nombre del Monumento
  3. - Vínculo a la página del CNM para el Monumento
  4. - Datos (dirección, descripción, url de una foto, y las coordenadas dadas para el monumento).

El archivo JSON proporcionado contiene la especificación CORS en sus headers; esto quiere decir que puede recibir requests desde cualquier origen.

Para más información sobre el desarrollo del API, puede encontrar el código en mi repo Git.

Ejemplo de utilización del API

El Mapa de la derecha es un ejemplo de aplicación en AngularJS que descarga el API, procesa y almacena aquellos monumentos que poseen coordenadas; luego muestra la información en una ventana al hacer click sobre el icono del monumento.

La aplicación de ejemplo es parte de un proyecto de aplicación móvil en Ionic/Cordova, que será publicada próximamente en la Play Store para Android.
